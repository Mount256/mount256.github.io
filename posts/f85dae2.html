1 如何在网页中获取 JSON 数据？

打开一个具有动态渲染的网页，按 F12 打开浏览器开发工具，点击“网络”，再刷新一下网页，观察是否有新的数据包。

发现有 js 后缀的文件，这就是我们想要的 json 数据了。

  • dumps：用于将 Python 的数据类型转化为 json 字符串。
